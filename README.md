


# AI-Integrated E-Commerce Platform

A modern AI-powered e-commerce platform developed using **Django**, **HTML**, **CSS**, and **JavaScript**.

This project is being developed as part of the **Community Based Training Program (CBTP) Phase II** for the Software Engineering Department at Jimma Institute of Technology, Jimma University.

---

## Project Overview

The platform provides a digital marketplace for local shops in Ginjo Guduru Kebele, allowing customers to browse products, place orders, and interact with AI-powered features such as customer support and sentiment analysis.
